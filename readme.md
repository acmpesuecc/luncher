## Getting Started
to get started, ensure
1. You have WSL (Windows Subsystem for Linux) installed with Ubuntu 20.04
2. Ensure you have docker, docker-compose and Git installed in WSL

## Running The Backend
* clone this repository
* type `docker-compose up`



## Stack
